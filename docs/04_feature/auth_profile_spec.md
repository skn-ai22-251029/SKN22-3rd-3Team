# 회원가입 및 프로필 관리 기능 명세

## 1. 개요
유저(집사) 맞춤형 서비스를 제공하기 위해 회원 정보와 반려묘 정보를 수집하고 관리하는 기능을 구현한다. 
초기 단계는 Mock-up 형태로 구현하여 데이터 흐름과 UI를 검증한다.

## 2. 데이터 수집 항목

### 2.1. 집사(User) 정보
서비스 이용 및 맞춤형 추천을 위한 기본 정보.
*   **계정 정보**:
    *   아이디 (ID)
    *   비밀번호 (Password) - *암호화 필수*
    *   (옵션) SSO 연동 (Google, Kakao 등)
*   **기본 프로필**:
    *   닉네임
    *   이름
    *   나이
    *   성별
    *   연락처
    *   주소
*   **양육 환경 정보** (기존 Onboarding 연계):
    *   거주 형태 (예: 아파트, 주택 등)
    *   일일 평균 부재 시간 (집을 비우는 시간)
    *   동거 가족 구성원
    *   고양이 양육 경험

### 2.2. 주인님(Cat) 정보
다묘 가정을 고려하여 여러 마리의 정보를 등록(Append)할 수 있어야 함.
*   **기본 정보**:
    *   이름
    *   나이
    *   성별
    *   **묘종**: 67종 리스트 중 선택 (`cat_breeds_integrated.json` / Atlas `catfit_v2.breeds` 기준)
    *   프로필 사진 (업로드)
*   **건강 정보**:
    *   마지막 접종일 (백신 종류 및 날짜)
    

## 3. 기능 시나리오 (Mock-up)

### 3.1. 회원가입/로그인
*   로그인 화면: ID/PW 입력 또는 소셜 로그인 버튼.
*   회원가입/Onboarding 화면: 단계별로 위 2.1, 2.2 정보를 입력받음.

### 3.2. 메인 대시보드 (접속 후 첫 화면)
수집된 데이터를 활용하여 개인화된 메시지 노출.
*   **인사 메시지**: "주인님, 어서 오십시오."
*   **알림 예시**:
    *   *"오늘 **레오**(고양이 이름)의 3차 접종 예정일이 1주일 남았습니다."*
    *   *"관련 상담을 연결할까요?"* (챗봇 연동 제안)
*   **가족 구성원 노출**: 다묘 집사를 위한 고양이 프로필 카드 리스트 등.
